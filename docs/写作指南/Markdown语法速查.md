# Markdown 语法速查

本文汇总了在 MDOnline 中可以使用的 Markdown 语法，方便随时查阅。

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 文本样式

| 语法 | 效果 | 快捷键 |
|------|------|-------|
| `**加粗**` | **加粗** | Ctrl+B |
| `*斜体*` | *斜体* | Ctrl+I |
| `~~删除线~~` | ~~删除线~~ | — |
| `==高亮==` | ==高亮== | — |
| `` `行内代码` `` | `行内代码` | — |

## 列表

### 无序列表

```markdown
- 第一项
- 第二项
  - 子项 A
  - 子项 B
- 第三项
```

### 有序列表

```markdown
1. 步骤一
2. 步骤二
3. 步骤三
```

### 任务列表

```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个待办
```

## 链接与图片

```markdown
[链接文字](https://example.com)
[带标题的链接](https://example.com "鼠标悬停提示")

![图片替代文字](图片地址.png)
![带标题的图片](图片地址.png "图片标题")
```

## 代码块

### 基础代码块

````markdown
```
这是普通代码块
```
````

### 带语法高亮

````markdown
```python
def hello():
    print("Hello, MDOnline!")
```
````

### 支持的语言

常见语言均可高亮：`javascript`、`python`、`java`、`go`、`rust`、`bash`、`css`、`html`、`json`、`yaml`、`sql` 等。

## 表格

```markdown
| 列1 | 列2 | 列3 |
|-----|:----:|----:|
| 左对齐 | 居中 | 右对齐 |
| 内容A | 内容B | 内容C |
```

效果：

| 列1 | 列2 | 列3 |
|-----|:----:|----:|
| 左对齐 | 居中 | 右对齐 |
| 内容A | 内容B | 内容C |

## 引用

```markdown
> 这是一段引用文字
>
> > 引用可以嵌套
```

> 这是一段引用文字
>
> > 引用可以嵌套

## 分割线

```markdown
---
```

## 脚注

```markdown
这是一个脚注[^1]

[^1]: 脚注的内容写在这里
```

## 数学公式

行内公式：`$E = mc^2$`，显示为 \(E = mc^2\)

块级公式：

$$
\sum_{i=1}^{n} x_i = x_1 + x_2 + \cdots + x_n
$$

> 数学公式渲染需要额外插件支持，MDOnline 默认不包含。

## 转义字符

使用反斜杠转义 Markdown 特殊字符：

```markdown
\*不是斜体\*
\# 不是标题
\[ 不是链接
```
