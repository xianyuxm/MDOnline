# 主题切换

MDOnline 内置**暗黑模式**和**浅色模式**双主题，支持一键切换并自动持久化用户偏好。

## 切换方式

点击页面右下角的浮动按钮即可切换：

| 当前模式 | 按钮图标 | 点击后 |
|---------|---------|-------|
| 暗黑模式 | ☀️ 太阳 | 切换为浅色模式 |
| 浅色模式 | 🌙 月亮 | 切换为暗黑模式 |

## 主题持久化

主题偏好自动保存到浏览器 `localStorage`，键名为 `docsify-theme`。

```
localStorage.getItem('docsify-theme')  // 返回 'dark' 或 'light'
```

下次访问时将自动应用上次的主题选择，无需重复设置。

## 技术实现

主题切换基于 CSS Variables 实现，核心变量定义在 `style.css` 中：

```css
/* 暗黑模式（默认） */
:root[data-theme="dark"] {
  --background: #1a1a2e;
  --text: #e0e0e0;
  --sidebar-bg: #16213e;
  /* ... */
}

/* 浅色模式 */
:root[data-theme="light"] {
  --background: #ffffff;
  --text: #333333;
  --sidebar-bg: #f5f5f5;
  /* ... */
}
```

## 自定义主题颜色

如果你想调整主题配色，只需修改 `style.css` 中对应的 CSS 变量值即可。

### 示例：将暗黑模式的主色调改为蓝色

```css
:root[data-theme="dark"] {
  --accent-color: #2f6df6;  /* 将绿色改为蓝色 */
}
```

## 注意事项

- 默认主题为**暗黑模式**（首次访问）
- 主题设置按浏览器独立存储，不会跨设备同步
- 清除浏览器数据会重置为主题默认值
