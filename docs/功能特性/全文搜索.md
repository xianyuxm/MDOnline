# 全文搜索

MDOnline 内置基于 Docsify 的全文搜索插件，支持中文分词，让你快速定位文档内容。

## 使用方式

在页面顶部搜索框中输入关键词，即可实时搜索所有文档内容。

- **搜索框位置**：侧边栏顶部
- **搜索快捷键**：按 `/` 键可快速聚焦搜索框
- **结果展示**：实时显示匹配的文档标题和内容片段

## 搜索配置

搜索功能在 `index.html` 中配置：

```javascript
window.$docsify = {
  search: {
    paths: 'auto',       // 自动搜索所有页面
    maxAge: 0,           // 不缓存（开发环境推荐）
    depth: 4,            // 搜索标题深度（到 h4）
    noData: '没有找到相关内容',
    placeholder: '搜索接口、流程、状态码'
  }
};
```

## 配置说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|-------|------|
| `paths` | String/Array | `auto` | 搜索路径，`auto` 自动扫描所有页面 |
| `maxAge` | Number | `86400000` | 搜索缓存时间（毫秒），设为 `0` 禁用缓存 |
| `depth` | Number | `6` | 搜索的标题层级深度 |
| `noData` | String | `'No Results'` | 无结果时的提示文字 |
| `placeholder` | String | `'Search'` | 搜索框占位文字 |

## 中文搜索

MDOnline 的搜索插件对中文有良好支持：

- 支持中文关键词精确匹配
- 支持中文标题搜索
- 支持中英混合搜索

> 搜索基于 Docsify 的 `search.min.js` 插件，该插件已内置中文分词支持。

## 性能优化建议

- **开发环境**：将 `maxAge` 设为 `0`，确保每次刷新都能搜到最新内容
- **生产环境**：将 `maxAge` 设为 `86400000`（1 天），利用缓存提升搜索速度
- **大量文档**：可通过 `paths` 指定只搜索关键目录，减少索引体积
