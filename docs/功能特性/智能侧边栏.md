# 智能侧边栏

MDOnline 通过 `gen_sidebar.py` 脚本自动生成侧边栏，让你专注写作，无需手动维护导航结构。

## 工作原理

1. 脚本扫描 `docs/` 下的所有子目录
2. 读取每个 `.md` 文件的一级标题（`# 标题`）作为链接文字
3. 按配置顺序生成 `_sidebar.md`
4. Docsify 加载 `_sidebar.md` 渲染侧边栏

## 使用方式

### 生成侧边栏

每次新增或删除 `.md` 文件后，运行：

```bash
python gen_sidebar.py
```

或使用一键启动脚本（会自动生成）：

```bash
start-server.bat
```

### 目录结构示例

```
docs/
├── 快速开始/
│   ├── 入门指南.md       # 标题：# 入门指南
│   └── 安装与部署.md     # 标题：# 安装与部署
├── 功能特性/
│   ├── 主题切换.md       # 标题：# 主题切换
│   └── 智能侧边栏.md    # 标题：# 智能侧边栏
└── 写作指南/
    └── Markdown语法速查.md
```

生成的侧边栏效果：

```
- 快速开始
  - 入门指南
  - 安装与部署

- 功能特性
  - 主题切换
  - 智能侧边栏

- 写作指南
  - Markdown语法速查
```

## 分组排序

在 `gen_sidebar.py` 顶部通过 `GROUP_ORDER` 控制分组的显示顺序：

```python
# 按此顺序排列，未列出的按字母序排在末尾
GROUP_ORDER = ['快速开始', '功能特性', '写作指南', '部署运维']
```

## 分组显示名

通过 `GROUP_NAMES` 将目录名映射为更友好的显示名：

```python
GROUP_NAMES = {
    '快速开始': '快速开始',
    '功能特性': '功能特性',
    '写作指南': '写作指南',
    '部署运维': '部署运维',
}
```

## 折叠功能

侧边栏分组支持点击折叠/展开，由 Docsify 插件自动实现：

- 点击分组标题可折叠/展开该分组
- 支持键盘操作（Enter / Space）
- 折叠状态不影响子页面内容

## 标题读取规则

- 脚本读取 `.md` 文件中的第一个 `# 一级标题` 作为链接文字
- 如果文件没有一级标题，则使用文件名（去掉 `.md` 后缀）
- 标题必须在文件开头，遇到非空非标题行后停止扫描

## 跳过文件

以下文件不会出现在侧边栏中：

| 文件 | 原因 |
|------|------|
| `_sidebar.md` | 侧边栏配置文件 |
| `_navbar.md` | 导航栏配置文件 |

如需跳过其他文件，在 `SKIP_FILES` 中添加：

```python
SKIP_FILES = {'_sidebar.md', '_navbar.md', '草稿.md'}
```
