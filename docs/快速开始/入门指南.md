# 入门指南

欢迎使用 **MDOnline** —— 一个零构建的 Markdown 文档站点解决方案。本指南将帮助你在 3 分钟内搭建起自己的文档站点。

## 前置条件

| 依赖 | 版本要求 | 用途 |
|------|---------|------|
| Python | ≥ 3.6 | 本地预览服务器 & 侧边栏生成 |
| 浏览器 | 现代浏览器 | 访问文档站点 |

> 无需 Node.js、Webpack、Vite 等前端工程化工具。

## 快速启动（3 步）

### 第 1 步：获取项目

将项目文件下载到本地任意目录。

### 第 2 步：启动服务器

**方式一：双击启动（推荐）**

双击项目根目录下的 `start-server.bat`，等待服务器启动完成。

> 如果双击运行失败，请使用方式二手动执行。

**方式二：手动启动**

在项目根目录打开命令行（CMD / PowerShell），依次执行：

```bash
py -m http.server 8080
```

> 如果 `py` 命令不可用，请尝试 `python` 或 `python3`。

### 第 3 步：访问站点

在浏览器中打开 [http://localhost:8080](http://localhost:8080)，即可看到文档站点。

## 目录结构总览

```
MDOnline/
├── index.html          # 入口文件（Docsify 配置）

├── vue.css             # Docsify 基础主题

├── docsify.min.js      # Docsify 核心
├── search.min.js       # 全文搜索插件
├── gen_sidebar.py      # 侧边栏自动生成脚本
├── start-server.bat    # Windows 一键启动
└── docs/               # 文档内容目录
    ├── 快速开始/
    ├── 功能特性/
    ├── 写作指南/
    └── 部署运维/
```
